1. Download weather and subway datasets from website.

2. Load the file from local file system to Dumbo:

scp weather.csv zh1272@dumbo.es.its.nyu.edu:/home/zh1272
scp subway.csv zh1272@dumbo.es.its.nyu.edu:/home/zh1272


3. Load the file to HDFS:

hdfs dfs -mkdir Spark_project
